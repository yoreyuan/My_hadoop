# 1 Git 谁与争锋
**版本控制系统**（Version Control System）VCS。

当前市场占用率最高的 VCS 有两套： Subversion（SVN）、Git。2013年以前 SVN 的市场占有率很高（40%以上），但至 2014 年后 Git 超越 SVN。
网上有专门提供 Git Server 服务的网站，比如 GitHub、GitLab。
